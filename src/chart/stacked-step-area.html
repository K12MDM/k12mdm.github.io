This sample visualizes the data about electricity production using stacked stepped area chart.

In this example, you can see how to render and configure the Stacked stepped type charts. This series forms the step line progress, by connecting points through vertical and horizontal lines with area filled. You can use dashArray, width, fill properties to customize the area. marker and dataLabel are used to represent individual data and its value.


More information on the StepArea series can be found in this documentation section.
